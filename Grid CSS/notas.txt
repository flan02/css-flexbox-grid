Para que sirve Grid y para que sirve Flexbox ?
    Grid se penso p/ armar, disenar los espacios de maquetacion para hacerlos con Grid areas,
    definimos asi las estructuras de la pagina web
    Flexbox alineamos los elem ya internos de tus componentes, ventana modal, elementos de tarjeta,

Son complementarios entre los 2.
